### 1. 英文关键词竞争态势（"Chinese astrology app" / "Ziwei Doushu app"）

- **[Google Play/App Store 紫微斗数相关App]**: 海外主流应用商店中，专门做"Ziwei Doushu"的英文应用极少。搜索"Chinese astrology"会大量返回星座（Western astrology）、印度占星（Hindu Jyotish）、塔罗类应用
- **"Ziwei Doushu" 英文专有名词几乎无市场认知** — 这是出海最大障碍之一，需要用"Chinese astrology"、"Four Pillars"、"BaZi"等更通用的词做SEO
- **竞品空缺**: 以"Ziwei Doushu"为精确命名的海外APP极少，部分是华人团队做的工具类应用，功能简单，无AI
- **可参考的英文竞品类型**:
  - "Chinese Astrology Pro"（Google Play）：评分4.6，用户约10万+，含命盘+流年
  - "Zuo King"（App Store）：有英文界面，但功能较基础，无深度AI解读
  - "Four Pillars of Destiny"（App Store）：BaZi（四柱八字）应用，界面西方化，覆盖面更广

### 口碑传播 vs 付费推广评估

**结论：早期阶段（0-3个月）口碑传播压倒性优于付费推广**

| 维度 | 口碑传播 | 付费推广 |
|------|---------|---------|
| 成本 | ⭐⭐（低）— 主要是人力/内容成本 | ⭐⭐⭐⭐⭐（高）— 信息流广告 ¥0.5-3/点击 |
| 用户信任度 | ⭐⭐⭐⭐⭐（高，朋友/博主推荐天然信任） | ⭐⭐⭐（中，需克服"广告"抵触心理） |
| 获客效率（早期） | ⭐⭐⭐⭐⭐（社交裂变可快速爆发） | ⭐⭐（低，产品未打磨时投放浪费大） |
| 可持续性 | ⭐⭐⭐⭐⭐（品牌积累，长期有效） | ⭐⭐⭐（停止投放即停止） |
| 数据可控性 | ⭐⭐⭐（相对难精确追踪） | ⭐⭐⭐⭐⭐（每分钱可追踪ROI） |
| 适合阶段 | 早期（0-6个月）+ 产品打磨期 | 中后期（有一定用户基数+成熟产品） |

**为什么口碑传播更适合早期：**
1. **命理产品天然适合社交传播**：用户算出有趣/震惊的命盘结论（如"你是天生的创业老板命"）有强烈分享欲，小程序内嵌"一键生成分享图"功能可大幅提升分享率
2. **内容营销成本低、复用率高**：一篇小红书爆款笔记可以持续带来流量，而信息流广告停止即消失
3. **付费推广在产品早期不划算**：转化漏斗未优化时，大量广告费浪费在产品体验不完善的环节
4. **口碑用户质量更高**：通过内容种草进来的用户通常已有使用意愿，付费用户多为冲动点击，付费率和复购率都更低

**付费推广适合的时机：**
- 当自然增长趋于平稳（DAU日增<10%）时
- 当产品核心体验已稳定（转化漏斗已优化）
- 当有清晰的用户画像和投放素材积累
- 推荐首先测试：小红书信息流广告（与内容种草联动）、微信朋友圈广告（定向30-40岁一二线城市用户）

**早期阶段推荐策略：口碑为主，付费为辅（试点）**
- 80%预算 → 内容营销（小红书笔记、公众号文章）
- 20%预算 → 小规模付费测试（每月¥500-1000信息流广告测试素材）
